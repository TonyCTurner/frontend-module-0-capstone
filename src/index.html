Why I love web development

I knew when I was 16 that I wanted to be an engineer. At the time, there was no such thing as the web. All that existed was primitive point to point connections between universities and military complexes.

History

While going through university, a new discipline was introduced, computer science. It was a very early version of what today is a vast science covering multiple areas of study. Back then, though, there was just Computer Science.

I built my first assembler, and watched as something I created in class take a file full of assembly language, which we also created, and assemble it into machine code. It was amazing to see the computer take this machine code, run it, and get a real output, from a language and an assembler that each of us invented out of thin air! I was hooked.

Very quickly, there came a plethora of languages, as folks experimented with new compilers and interpreters

  1. Forth
  2. Pascal
  3. ADA
  4. various assembly languages for different hardware platforms such as the 8080 and the motorolla chipsets
  5. Then, along came C
  6. And not too long afterwards, a very strange language that used a remarkable new technology, object oriented c++

Software was never the same. Procedural languages fell by the wayside like Pascal, replaced by object oriented Pascal.

I remember moving to Colorado, and working in my office in the basement. I became aware of technology advancements that created this strange thing called the World Wide Web. I began using it, and fell in love with static html. what an amazing thing. Soon came:

  • css
  • and then a strange language called Javascript

and the web came alive. Internet speeds were phenomenal, finally moving from 9600 baud to 19.2k baud! we were all amazed by this blinding speed. Harddrives grew to a wopping 10M, who would ever need more? Then, it happened:

  • DSL was born
  • CPUs doubled in speed
  • the first 100M harddrive came out for a mere $2000

, and off we went.

Today

It has been many years since then, and I find I still can't get enough of it. New technologies come out all the time, moving the art forward inch by inch.

Nowadays, I have a github account and I'm working through the entire Turing course before and after work.
